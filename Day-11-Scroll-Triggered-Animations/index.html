How it works

3 simple steps

1️⃣

Add a class

Give any element a class like fade-up, fade-left, or zoom-in. It starts hidden.

2️⃣

IntersectionObserver watches it

JavaScript uses IntersectionObserver to detect when the element enters the viewport.

3️⃣

.visible is added

When visible, JS adds the .visible class. CSS transitions do the rest — no extra JS needed.
