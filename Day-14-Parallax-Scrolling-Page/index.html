Parallax
Scrolling

The background moves at a different speed than the content — creating an illusion of depth as you scroll.

↓ scroll

How Does It Work?

As you scroll, JavaScript reads window.scrollY and moves each background layer at a fraction of that speed using translateY.

📜

Read Scroll Position

window.scrollY gives the number of pixels scrolled from the top of the page.

✖️

Multiply by Speed

Multiply scrollY by a value like 0.4. The background moves 40% as fast as you scroll.

🎯

Apply with translateY

Set style.transform = translateY(offset + 'px') on the background element every frame.
